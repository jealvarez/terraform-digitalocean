# **Droplet**

## Requirements

- [DigitalOcean](http://digitalocean.com/) Account
  - DigitalOcean Token
  - Spaces Access Keys
  - Created Space
- Terraform v0.11.7+

## Steps

- Execute `init.sh` script

```bash
./init.sh
```

- If you want to know which commands were executed, see `output.log` file
